Selamat datang di Dunia Fantasi

Book of fantasia adalah sebuah website khusus bagi anda yang gemar membaca dan suka berimajinasi, cikal bakal bagi penulis ataupun pengarang cerita yang ingin mencari inspirasi. Didasarkan oleh komentar berbagai pihak dan interpretasi pemilik website menjadi bahan yang kami sediakan sebagai bacaan anda.dapat berupa mitologi kuno sejarah kisah nyata, ataupun hanya animasi, semua hal yang berbau fantasi akan dibahas dalam website ini.

Latar belakang website

Dibuat oleh winto junior khosasih untuk menjadi tempat menuangkan imajinasi dan kreativitas seseorang. Karena beliau percaya bahwa ilmu pengetahuan pertama sekali datang dari imajinasi/mengira-ngira lalu berani mencoba dan mencari tahu. Dalam kesempatan ini beliau juga melatih diri dalam pembuatan website yang dinamis serta bermanfaat.

Content dari website ini


Community dan forum

Pada community dan forum website ini terdapat dua jenis tipe diskusi, pribadi dan umum navigasinya dapat anda coba sendiri.

Menu login dan submit

Sebelum anda dapat menggunakan community atau forum anda harus terlebih dahulu mendaftarkan diri. Untuk navigasi submit hanya bisa untuk user berizin dari administrator.

Book list interface

Didasari oleh searching field database dan mode reading PDF pada masing masing kapabilitas web browser..

Penjelasan cara menggunakan website

Sebagai guest anda dapat membaca dan mengakses list buku, namun untuk dapat sharing pendapat dan berbicara dalam komunitas kami, untuk itu anda harus membuat akun terlebih dahulu.

Anda juga dapat mensubmit / mengirim file karangan anda sendiri untuk menambah koleksi buku kami, namun untuk itu anda harus mendapat izin dari admin. Agar dapat berbicara dengan admin, anda harus bergabung pada community.

Beberapa fitur website mungkin akan kurang lengkap ataupun tidak ada, kalaupun sudah ada tapi anda merasa kurang nyaman dengan pengembangan dari admin, jangan malu untuk share dikomunitas kami. Admin akan mempertimbangkan perubahan konten website berdasarkan saran anda
